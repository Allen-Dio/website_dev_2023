    distance(a, b)

    Calculates the linear distance on a geometric ring from 0 to 2^256 any distance. Because it's a ring the maximal distance is two points being opposite of each other. In that case the distance is 2^255.

    2^256 = 0xffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffff 2^255 = 0x7fffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffff
