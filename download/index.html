Unzip and install

On Linux and OSX you can use the unzip terminal tool. On Windows you can double click it.

$ unzip Downloads/diode.zip We recommend moving it into a $PATH directoy such as /usr/local/bin or ~/bin if used.

Setup your fleet

To use Diode you have to specify which fleet you're running on.

$ diode config -set fleet=<YOUR_FLEET>

Publish ports

To publish e.g. your local http ports to the Diode Network call:

$ diode publish -public 80:80
