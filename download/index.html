Unzip and Install

Unzip the terminal tool. On Windows you can double click it.

$ unzip Downloads/diode.zip We recommend moving it into a $PATH directory such as /home/[user]/.local/bin

Publish Ports

To publish e.g. your local http ports to the Diode Network call:

$ diode publish -public 80:80
